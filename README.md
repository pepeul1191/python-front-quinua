## Gestion Flask Python

Requisitos de software previamente instalado:

	+ Python 3.5
	+ Python PIP

### Descipción

Instalación de dependencias:

    $ sudo pip install virtualenv
    $ virtualenv -p python3 <<nombre_ambiente>>
    $ source bin/activate
    $ pip install -r requirements.txt
